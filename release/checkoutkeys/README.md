# License Keys Manager

WordPress plugin for [CheckoutKeys.com](https://checkoutkeys.com) - automatically generate and manage license keys for your Stripe Checkout payments.

## Requirements

- WordPress 5.8+
- PHP 7.4+
- CheckoutKeys.com account

## Installation

```bash
cd wp-content/plugins
git clone https://github.com/checkoutkeys/license-keys-manager.git checkoutkeys
```

Activate the plugin through WordPress admin.

## Setup

1. Get your API key from [CheckoutKeys.com dashboard](https://checkoutkeys.com/dashboard)
2. In WordPress, go to **License Keys → Settings**
3. Enter your API key and save

## Features

- Sync licenses from CheckoutKeys.com
- View and manage all license keys
- Activate/deactivate licenses
- Copy license keys to clipboard
- Track activity logs

## Usage

Go to **License Keys → License Keys** and click "Sync from checkoutkeys.com" to import your licenses.
