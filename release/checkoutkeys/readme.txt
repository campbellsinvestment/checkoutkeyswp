== Description ==

CheckoutKeys License Manager is the official WordPress plugin for CheckoutKeys.com, a license key management service integrated with Stripe. This plugin allows you to sync license keys from CheckoutKeys.com, view subscription status, track activations, monitor plan limits, and manage everything from WordPress.

= Features =

* Automatic Sync: Sync all your license keys from CheckoutKeys.com with one click
* Dashboard Overview: See total, active, and recent licenses at a glance
* Plan Management: View your current plan, usage, and limits
* WordPress Native UI: Clean, familiar WordPress admin interface
* Secure API Integration: Uses Bearer token authentication
* Real-time Data: Fetches subscription status directly from CheckoutKeys.com

= How It Works =

1. Sign up at CheckoutKeys.com
2. Get your API key from your CheckoutKeys dashboard
3. Install this plugin and enter your API key
4. Sync your license keys with one click
5. Manage everything from WordPress

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/checkoutkeys` directory, or install through WordPress plugins screen
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to CheckoutKeys â†’ Settings
4. Enter your API key from CheckoutKeys.com
5. Click "Sync from checkoutkeys.com" to import your license keys

== Frequently Asked Questions ==

= Do I need a CheckoutKeys.com account? =

Yes, you need an active account at CheckoutKeys.com to use this plugin.

= Is there a free plan? =

Yes, CheckoutKeys.com offers a free plan with up to 10 license keys.

= How often should I sync? =

The dashboard shows real-time subscription data automatically. For license keys, you can sync manually as needed.

= Where is my data stored? =

License keys are stored in your WordPress database. Subscription and plan data is fetched in real-time from CheckoutKeys.com.

= Is it secure? =

Yes, the plugin uses secure Bearer token authentication and follows WordPress security best practices.
